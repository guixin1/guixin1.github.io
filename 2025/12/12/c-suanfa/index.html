sort中比较函数

sort(起始位置, 结束位置, 比较规则);

1. 起始位置 (Iterator/Pointer)

  • 代码对应：arr + 1
  • 格式含义：排序区间的起点（包含该位置的元素）。
  • 注意：因为你写的是 arr + 1，说明你的数据是从数组下标 1 开始存放的（放弃了下标 0），这在算法竞赛中很常见。

2. 结束位置 (Iterator/Pointer)

  • 代码对应：arr + n + 1
  • 格式含义：排序区间的终点（不包含该位置的元素）。
  • 原理：C++ 的 STL 都是左闭右开区间 [begin, end)。如果要排序 n 个元素。起点是 1。终点必须是 n + 1，这样处理的范围才是 1 到 n。

3. 比较规则 (Comparator)

  • 代码对应：[&](const Q& x, const Q& y) { … }
  • 格式含义：这是一个 Lambda 表达式（匿名函数）。如果不写这个参数，默认是升序（从小到大）。写了就是自定义排序。
  • Lambda 的语法格式分解：codeText[捕获列表] (参数列表) { 函数体 }[&] (捕获列表)：表示函数体内部可以按引用访问外部所有的变量（虽然在这个简单的比较逻辑里通常用不到外部变量，但写上 & 或 = 是为了方便）。(const Q& x, const Q& y) (参数列表)：接收两个待比较的元素 x 和 y。const Q& 表示以常量引用传递，既避免了拷贝结构体的开销（速度快），又保证不会修改数据（安全）。{ … } (函数体)：返回类型必须是 bool。规则：如果你希望 x 排在 y 前面，就返回 true。
